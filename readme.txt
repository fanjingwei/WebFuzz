本项目仅在win10和win7下测试过，使用的python版本为3.5.2版本，依赖WFuzz项目，可以使用pip install wfuzz进行安装，版本为3.1.0：
项目地址：https://github.com/xmendez/wfuzz/
WFuzz项目帮助文档地址：https://wfuzz.readthedocs.io/en/latest/user/basicusage.html?highlight=-H

如果你不幸遇到了pycurl错误，请到：https://www.lfd.uci.edu/~gohlke/pythonlibs/下载pycurl-7.43.0.3-cp35-cp35m-win_amd64.whl，然后通过pip install pycurl-7.43.0.3-cp35-cp35m-win_amd64.whl安装pycurl解决问题
